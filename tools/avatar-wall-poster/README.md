# 头像墙海报生成器

这是一个独立的小工具，用纯前端 HTML/CSS/JavaScript 实现，不依赖常州 AI Club 官网、Next.js、Supabase 或任何构建步骤。

## 能做什么

- 上传微信群成员长截图，自动裁切头像并生成编号校验图
- 用头像填充 `51`、`61`、文字或其他数字形状
- 使用五一、六一等节日预设，也可以自定义标题、口号、品牌与配色
- 导出透明头像墙 PNG、完整海报 PNG、头像编号图
- 上传自己的无字背景，或用本地 Canvas 生成节日底图

## 本地运行

可以直接打开 `index.html`。如果浏览器限制本地文件读取，进入本目录运行：

```bash
npm run dev
```

然后访问：

```text
http://127.0.0.1:5174
```

## 设计思路

这版刻意把「背景」「文字」「头像墙」拆成三层：

1. 背景尽量保持无字，方便用 AI 图或手工设计图替换。
2. 标题、口号、品牌信息由工具本地绘制，避免生成图里的错字。
3. 头像墙单独可导出透明 PNG，方便继续拿去 PS、Canva 或脚本二次合成。

六一海报实践里发现，头像外层轮廓线容易压住边缘头像，所以默认关闭上方轮廓线，只保留头像白色描边和底层数字轮廓。
