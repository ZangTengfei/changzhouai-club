# 常州 AI Club 文档

这里是社区的内容沉淀区，适合放：

- 活动组织 SOP
- 主题分享讲稿与复盘
- 项目共建流程
- 成员实践经验

## 这套文档适合怎么用

### 1. 快速了解社区如何协作

先阅读 [开始使用](/docs/getting-started) 和 [参与贡献](/docs/contributing)，就能知道这里的内容范围、更新方式和共建约定。

### 2. 逐步沉淀真实经验

和“宣传型页面”不同，这里更适合记录真实工作流、踩坑、模板和复盘。内容越贴近真实实践，后续复用价值越高。

### 3. 用 Git 和 PR 做共建

我们先采用最轻量的共建方式：内容直接进入当前仓库，通过 PR 审阅和迭代。这样主站和文档可以一起演进。
